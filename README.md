
# Company Knowledgebase Chatbot

A simple RAG chatbot built using:

- Streamlit
- LangChain
- LangGraph
- FAISS
- OpenAI

## Setup

1. Install dependencies:

pip install -r requirements.txt

2. Add API key in `.env`

3. Put PDFs in `data/`

4. Run ingestion:

python ingest.py

5. Run app:

streamlit run app.py
